Arrays

In this module I will talk about arrays. An Array is a numbered list of strings. Arrays are the best way to hold multiple sting elements too. The easiest way to create a simple array is to use the () syntax. This syntax is a good way to creat an array with static data. Now that an array is created you can use $ with the name of the array with other commands like echo to in this if you don't specify it will print the first value on your array.
